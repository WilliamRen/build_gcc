# gcc.sh
you can build a native compilr.
See help `./gcc.sh -h`

    $ chmod +x gcc.sh
    $ ./gcc.sh

Tested on

- OS X 10.8.2
- CentOS 5.8 x86_64


# gcc-mingw-w64.sh
you can build a cross Win32 and Win64 compiler.
See help `./gcc-mingw-w64.sh -h`

    $ chmod u+x gcc-mingw-w64.sh
    $ ./gcc-mingw-w64.sh

Tested on

- OS X 10.8.2
- CentOS 5.8 x86_64
- Cygwin 1.7.17-1
